1. Прерывание цикла: «break»

Обычно цикл завершается при вычислении условия в false

Возможно также выйти из цикла в любой момент с помощью специальной директивы break

См. код

Сочетание «бесконечный цикл + break» - применяется для тех ситуаций, когда условие, по которому нужно прерваться, находится не в начале или конце цикла, а посередине или даже в нескольких местах его тела

2. Переход к следующей итерации: continue

Директива continue - «облегчённая версия» break

При её выполнении цикл не прерывается, а переходит к следующей итерации (если условие все ещё равно true)

Директива continue позволяет избегать вложенности, что улучшает читабельность кода

3. Метки для break/continue

Если нужно выйти одновременно из нескольких уровней цикла сразу, применяются метки

Синтаксис:

labelName: for (...) { ... }

См. код

Порядок выполнения

Вызовом break outer будет разорван внешний цикл до метки с именем outer

Управление перейдёт со строки, помеченной (*), к alert('Готово!')

Можно размещать метку на отдельной строке:

outer:
for (let i = 0; i < 3; i++) { ... }

Директива continue также может быть использована с меткой

В этом случае управление перейдёт на следующую итерацию цикла с меткой

Метки не позволяют «прыгнуть» куда угодно

Метки не дают возможности передавать управление в произвольное место кода

Директива break должна находиться внутри блока кода

Технически, подойдет любой маркированный блок кода

Итого

while - Проверяет условие перед каждой итерацией

do..while - Проверяет условие после каждой итерации

for (;;) - Проверяет условие перед каждой итерацией, есть возможность задать дополнительные настройки

Бесконечный цикл использует конструкцию while (true)

Любой цикл может быть прерван директивой break

Если на данной итерации цикла делать больше ничего не надо, но полностью прекращать цикл не следует - используют директиву continue

Директивы break/continue поддерживают метки, которые ставятся перед циклом

Метки - единственный способ для break/continue выйти за пределы текущего цикла, повлиять на выполнение внешнего
