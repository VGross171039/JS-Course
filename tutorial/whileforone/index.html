Циклы while и for, часть 1

  • Страница урока

Рассматриваются базовые циклы: while, do..while и for()

Для многократного повторения одного участка кода предусмотрены циклы

1. Цикл «while»

Общий синтаксис: while (condition) { code }

Код из тела цикла выполняется, пока условие condition истинно

Одно выполнение тела цикла называется итерация

Любое выражение или переменная может быть условием цикла

См. код

Фигурные скобки не требуются для тела цикла из одной строки

while (i) alert (i--)

2. Цикл «do … while»

Проверку условия можно разместить под телом цикла, используя специальный синтаксис do..while

Цикл сначала выполнит тело, а затем проверит условие condition, и будет выполняться пока его значение равно true

Такая форма синтаксиса оправдана, если нужно чтобы тело цикла выполнилось хотя бы один раз, даже если условие окажется ложным

3. Цикл «for»

Более сложный, но при этом самый распространённый цикл

Составные части

Начало let i = 0 Выполняется один раз при входе в цикл

Условие i < 3 Проверяется перед каждой итерацией цикла. Если оно вычислится в false, цикл остановится

Тело alert(i) Выполняется снова и снова, пока условие вычисляется в true

Шаг i++ Выполняется после тела цикла на каждой итерации перед проверкой условия.

Порядок выполнения

Выполнить начало

→ (Если условие == true → Выполнить тело, Выполнить шаг)

→ (Если условие == true → Выполнить тело, Выполнить шаг)

→ (Если условие == true → Выполнить тело, Выполнить шаг)

→ (Если условие == false → Завершить цикл)

Начало выполняется один раз
Каждая итерация заключается в проверке условия, после которой выполняется тело и шаг

Обьявление переменной внутри счетчика называется «встроенным» объявлением переменной

Такие переменные существуют (видны) только внутри цикла
