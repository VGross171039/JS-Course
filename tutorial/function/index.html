1. Общая информация

Для повторения одного и того же действия во многих частях программы служат функции

Функции являются основными строительными блоками программы

Примеры встроенных функций: alert(message), prompt(message, default) и confirm(question)

Пользовательские функции создаются программистом самостоятельно в зависимости от конкретных задач

2. Объявление функции

Для создания функций используется объявление функции

Вначале идёт ключевое слово function, после него имя функции

Затем список параметров в круглых скобках через запятую

При отсутствии установленных параметров круглые скобки пустые

Код функции, также называемый «телом функции», внутри фигурных скобок

Синтаксис

function functionName(params) { ...code... }

Функция может быть вызвана по своему имени: functionName(params)

Вызов functionName(params) выполняет код функции

Локальные переменные

Переменные, объявленные внутри функции, видны только внутри этой функции

См. код

Внешние переменные

У функции есть доступ к внешним переменным

Функция может изменять значение внешних переменных

См. код

Внешняя переменная используется, только если внутри функции нет такой локальной

Глобальные переменные

Переменные, объявленные снаружи всех функций, называются глобальными

Глобальные переменные видимы для любой функции (если только их не перекрывают одноимённые локальные переменные)

Желательно сводить использование глобальных переменных к минимуму. В современном коде обычно мало или совсем нет глобальных переменных. Хотя они иногда полезны для хранения важнейших «общепроектовых» данных
