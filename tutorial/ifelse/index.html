Условное ветвление: if, '?'

Для выполнения различных действий в зависимости от условий, мы можем использовать инструкцию if и условный оператор ?, который также называют оператором «вопросительный знак»

1. Инструкция «if»

Инструкция if(...){code} вычисляет условие в скобках и, если результат true, то выполняет блок кода

Рекомендуется использовать фигурные скобки всегда, когда используется инструкция if, даже если выполняется только одна команда

2. Преобразование к логическому типу

Число 0, пустая строка "", null, undefined и NaN в круглых скобках - становятся false

Код при таком условии никогда не выполнится

Остальные значения становятся true

Код при таком условии выполняется всегда
