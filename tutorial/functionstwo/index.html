Возврат значения

Функция может вернуть результат, который будет передан в вызвавший её код

Для этого применяется директива return

Директива return может находиться в любом месте тела функции

Как только выполнение доходит до этого места, функция останавливается

При этом значение возвращается в вызвавший её код (присваивается переменной result, например)
