Оператор нулевого слияния (??)

Эта возможность была добавлена в язык недавно. В старых браузерах может понадобиться полифил

Представляет собой два вопросительных знака ??

Обрабатывает null и undefined одинаковым образом

Значение «определено», если оно не равняется ни null, ни undefined

Результат выражения a ?? b

- Если a определено, то a

- Если a не определено, то b

Оператор ?? возвращает первый аргумент, если он не null/undefined, иначе второй

Аналогии

result = a ?? b

result = (a !== null && a !== undefined) ? a : b;

1. Сравнение с ||

Оператор ИЛИ || можно использовать для того же, что и ??

Принципиальное различие между ними:

|| возвращает первое истинное значение

?? возвращает первое определённое значение

3. Использование ?? вместе с && или ||

По соображениям безопасности JavaScript запрещает использование оператора ?? вместе с && и ||, если приоритет явно не указан при помощи круглых скобок

let x = 1 && 2 ?? 3

Код выше вызовет ошибку

Рекомендуется использовать скобки, чтобы обойти это ограничение

Итого

Оператор нулевого слияния ?? — это быстрый способ выбрать первое «определённое» значение из списка.

Используется для присвоения переменным значений по умолчанию

Оператор ?? имеет низкий приоритет, поэтому при использовании его в выражении, скорее всего, потребуются скобки

Запрещено использовать вместе с || или && без явно указанного приоритета, то есть без скобок
