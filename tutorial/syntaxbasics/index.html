1. Привет мир

Для добавления кода JavaScript на страницу используется тег script

Атрибуты type и language необязательны

Предпочтительнее внешнее подключение скриптов пакетами

2. Структура кода

Инструкции – это синтаксические конструкции и команды, которые выполняют действия

Для читабельности каждую инструкцию пишут на новой строке

Рекомендуется разделять инструкции точкой с запятой, даже если они отделены переносами строк

При написании кода рекомендуется добавляьь комментарии, которые бы описывали, что делает код и почему

Однострочные комментарии начинаются с двойной косой черты //

Часть строки после // считается комментарием

Многострочные комментарии начинаются косой чертой со звёздочкой /* и заканчиваются звёздочкой с косой чертой */

Содержимое комментария игнорируется

Вложенные комментарии не поддерживаются!

3. Строгий режим — "use strict"

Директива выглядит как строка: "use strict" или 'use strict'

Иницирует работу скрипта в современном режиме

Должен находится в первой исполняемой строке скрипта

Над "use strict" могут быть записаны только комментарии

Нет никакого способа отменить use strict

Очень желательно добавлять "use strict" в начале всех скриптов
