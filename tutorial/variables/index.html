1. Общее описание

Переменная - это «именованное хранилище» для данных

Для создания переменной в JavaScript используется ключевое слово let

Переменная может быть объявлена только один раз. Повторное объявление вызывает ошибку

Возможно изменение значения переменной

Возможно объявить две переменные и скопировать данные из одной в другую

Функциональные языки программирования Scala или Erlang запрещают изменять значение переменной

Данные языки рекомендуются для ознакомления

ScalaThe Scala Programming Language

ErlangPractical functional programming for a parallel world

1. Имена переменных

Имя переменной должно содержать только буквы, цифры или символы $ и _

Первый символ не должен быть цифрой

Если имя содержит несколько слов, обычно используется верблюжья нотация (CamelCase)

Знак доллара '$' и подчёркивание '_' также можно использовать в названиях

Регистр написания имеет значение

Нелатинские буквы разрешены, но не рекомендуются

Использование зарезервированных имен вызывает ошибку

2. Константы

Переменные, объявленные с помощью const, называются «константами»

Константу нельзя изменить. Попытка сделать это приведёт к ошибке

Константы в верхнем регистре

Рекомендуется использование констант в качестве псевдонимов для трудно запоминаемых значений, которые известны до начала исполнения скрипта

Названия таких констант пишутся с использованием заглавных букв и подчёркивания

Правила применения заглавных и обычных букв в константах

Константы, известные до выполнения скрипта - обозначаются заглавными буквами

Константы, вычисляемые во время выполнения сценария, но не изменяющиеся после их первоначального назначения - обозначаются обычными буквами

Придумывайте правильные имена

Используйте легко читаемые имена, такие как userName или shoppingCart

Избегайте использования аббревиатур или коротких имён

Делайте имена максимально описательными и лаконичными

Договоритесь с вашей командой об используемых терминах

Итого

Мы можем объявить переменные для хранения данных с помощью ключевых слов var, let или const

let - это современный способ объявления

var - это устаревший способ объявления

const - похоже на let, но значение переменной не может изменяться

Переменные и константы должны быть названы таким образом, чтобы было легко понять, что у них внутри
